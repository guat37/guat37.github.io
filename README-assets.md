# Assets CV en ligne

Copier ces fichiers dans `assets/images/` du dépôt Jekyll.

- `greg-hero-executive.webp` : visuel principal Hero.
- `greg-about-editorial.webp` : portrait section Vision / À propos.
- `greg-wide-cinematic.webp` : visuel cinématographique pleine largeur.
- `topographic-lines.svg` : motif de lignes topographiques, à utiliser en fond CSS/SVG.

Des variantes redimensionnées sont fournies pour les balises `<picture>` ou `srcset`.
